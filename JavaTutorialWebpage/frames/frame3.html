Enkapsulacja (hermetyzacja)

Hermetyzacja jest mechanizmem służącym do ochrony danych przed nieuprawnionym dostępem lub modyfikacją. W Javie hermetyzacja jest osiągana poprzez użycie modyfikatora dostępu private. 

public class Obliczenia {
    private double liczba1;
    private double liczba2;

    public Obliczenia(double liczba1, double liczba2) {
        this.liczba1 = liczba1;    
        this.liczba2 = liczba2;
    }

    public double getSuma() {
        return liczba1 + liczba2;
    }
}

W powyższym przykładzie zmienne liczba1 i liczba2 są hermetyzowane poprzez użycie modyfikatora private.
Aby mieć dostęp do tych zmiennych z innych klas, można stworzyć metody getLiczba1() i getLiczba2().
Aby zmienić wartości tych zmiennych, można stworzyć metody setLiczba1() i setLiczba2(). 

public double getLiczba1() {
    return liczba1;
}
public double getLiczba2() {
    return liczba2;
}

public void setLiczba1(double liczba1) {
    this.liczba1 = liczba1;
}
public void setLiczba2(double liczba2) {
    this.liczba2 = liczba2;
}
